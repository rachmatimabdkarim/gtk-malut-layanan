📋 Platform yang didukung: Google Drive/Docs/Sheets/Slides, Dropbox, OneDrive, Box, MEGA

🔓 Penting: Pastikan file diatur "Anyone with the link can view"

📖 Cara mengatur akses publik

Google Drive/Docs:

1. Klik kanan file → Share → Change to "Anyone with the link"

2. Set permission ke "Viewer" → Copy link

Dropbox:

1. Klik Share → Create link → Copy link

OneDrive:

1. Klik Share → Anyone with the link can view → Copy link
